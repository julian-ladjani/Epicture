showSeekBarValue

static val showSeekBarValue: Int
static val showSeekBarValue: Int

Flag indicating whether the TextView next to the seekbar that shows the current seekbar value will be displayed. If true, the view is VISIBLE; if false, the view will be GONE. By default, this view is VISIBLE.

May be a boolean value, such as "true" or "false".
