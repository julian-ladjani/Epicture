Attributes that can be used with a AppCompatTextView.

Includes the following attributes:

AttributeDescription android:textAppearance com.appdev.epitech.epicture:autoSizeMaxTextSizeThe maximum text size constraint to be used when auto-sizing text. com.appdev.epitech.epicture:autoSizeMinTextSizeThe minimum text size constraint to be used when auto-sizing text. com.appdev.epitech.epicture:autoSizePresetSizesResource array of dimensions to be used in conjunction with autoSizeTextType set to uniform. com.appdev.epitech.epicture:autoSizeStepGranularitySpecify the auto-size step size if autoSizeTextType is set to uniform. com.appdev.epitech.epicture:autoSizeTextTypeSpecify the type of auto-size. com.appdev.epitech.epicture:firstBaselineToTopHeightDistance from the top of the TextView to the first text baseline. com.appdev.epitech.epicture:fontFamilyThe attribute for the font family. com.appdev.epitech.epicture:lastBaselineToBottomHeightDistance from the bottom of the TextView to the last text baseline. com.appdev.epitech.epicture:lineHeightExplicit height between lines of text. com.appdev.epitech.epicture:textAllCapsPresent the text in ALL CAPS.
